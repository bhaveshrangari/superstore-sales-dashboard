# Superstore-Sales-Dashboard
📌 Overview
This project showcases a comprehensive Power BI dashboard analyzing Superstore sales data across multiple dimensions — including time, geography, product categories, customer segments, and shipping preferences.

🎯 Objectives
Visualize key performance indicators (KPIs) like Total Sales, Profit, and Quantity Sold

Identify top-performing segments, states, and product categories

Compare year-over-year trends to uncover seasonal patterns

Analyze shipping methods and payment modes for operational insights

📊 Dashboard Features
KPI Cards: Total Sales, Profit, Quantity

Segment & Payment Analysis: Profit breakdown by customer segment and payment mode

Monthly Trends: Year-over-year comparison of sales and profit

Shipping Class Insights: Sales contribution by shipping method

Geographic Map: State-wise performance visualization

Category/Subcategory Breakdown: Deep dive into product performance

🧠 Key Insights
Consumer segment leads in profitability

COD surprisingly outperforms other payment modes

California and Texas dominate in sales and profit

Office Supplies and Phones are top-selling categories

Standard Class shipping is most preferred

🛠️ Tools Used
Power BI for data modeling and visualization

Excel/CSV for data preprocessing

DAX for calculated measures and dynamic filtering
